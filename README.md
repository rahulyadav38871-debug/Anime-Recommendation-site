# Anime Recommendation Site (AniMatch)

Final Year Project frontend built with Next.js + TypeScript + Tailwind.

## Features
- Discover / Search anime UI
- Anime details page
- Watchlist / Favorites UI
- Profile and onboarding flows

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui

## Run locally
npm install
npm run dev
